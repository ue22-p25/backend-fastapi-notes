# A Notes App in FastAPI - step by step

## Purpose

This repo describes how to build **a notes app with FastAPI**, from scratch, and
step by step.  
Each step outlines a specific feature, and how to implement it.

It's probably best not to try and implement the code yourself, since that would
drain your bandwidth, and in any case the final code is available in the repo below;
instead, you can read through the code and comments, and try to figure out how it
works.
